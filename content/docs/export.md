# Export

## Settings and export

The assistant can **read export settings**, **change a setting**, **start export** with options, or **export now** with the current UI settings. Heavy exports are still bound by your CPU, GPU, and disk.

```text
What are my current export settings?
```

```text
Set export resolution to 1080p and start export to my Videos folder.
```

```text
Export now with the same settings as last time.
```

Large projects may block other work while encoding. Let long jobs finish before you start another GPU heavy generation task.

> **Tip:** Save the project before a long export. If the app queues heavy tasks, exports may wait their turn behind other jobs.
