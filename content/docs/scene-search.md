# Scene search and smart cuts

## Scene search on a selected clip

The assistant can **search the selected clip** for scenes that match a text query (what you would say out loud: “the moment they open the door”, “laughing in the kitchen”). Results come back as candidate moments the tool can work with.

Example:

```text
Search the selected clip for the first clear shot of the product box.
```

## Slice at best match

After a search, you can ask to **slice at the best match** so the timeline jumps to a sensible cut point near the strongest hit. Phrase it as an outcome:

```text
Cut the selected clip at the best match for "door opening".
```

> **Tip:** Select the right clip on the timeline before you ask. If nothing matches, loosen the query or scrub manually once to give the model a better verbal anchor.
