# Director analysis

These tools are **read only**. They inspect the current project and return structured commentary: pacing, audio levels, transition patterns, clip content summaries, music sync hints, and similar. They do not change the timeline until you ask for separate edit actions.

## What you can ask

```text
Analyze the timeline structure and call out any overloaded segments.
```

```text
How are audio levels across the main dialogue track?
```

```text
Do transitions cluster too much in the first minute?
```

```text
Summarize what happens in the selected clip visually.
```

Use the answers to decide your next **concrete** edit requests (cuts, levels, new transitions).

> **Tip:** Run analysis after a big import or reorder, not before, so the picture matches the timeline you actually have.
