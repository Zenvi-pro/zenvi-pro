# Open source

Zenvi builds on a long lineage of open video software (see the **zenvi-core** README for license and upstream history). We are bootstrapped, but we believe **serious tools should be inspectable** and that community contributions can make the editor better for everyone.

## Repository

Source and release automation for the desktop core live here:

[github.com/Zenvi-pro/zenvi-core](https://github.com/Zenvi-pro/zenvi-core)

## Build your own flavor

You may fork, patch, and ship for internal use within the terms of the license. Respect trademark and branding if you redistribute binaries to the public so users are not confused about who supports their build.

Practical steps:

1. Clone **zenvi-core**.
2. Follow **README** for **libopenshot** and Python deps.
3. Run from source with the documented scripts.
4. Join issues and discussions when you want to upstream fixes.

[Install from source](/docs/install) (overview on this site)

> **Tip:** If you only need plugins or scripts, consider starting with **issues** and small PRs before maintaining a full fork.
