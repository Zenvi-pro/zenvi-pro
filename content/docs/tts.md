# Text to speech

The assistant can add **TTS audio** to the timeline as its own tool. Describe the voice style in everyday words and where it should go.

```text
Add TTS that reads "Subscribe for weekly tips" in a calm neutral voice at the playhead.
```

Length and voice options depend on the engine wired into your build. If TTS is not configured, the tool will return an error you can read in the chat thread.

> **Tip:** Export a scratch track first at low priority if you are experimenting with wording, then regenerate once the script is final.
