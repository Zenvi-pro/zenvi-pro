

# Zenvi.pro - AI Video Editor Landing Page & App

## Overview
A stunning dark-mode landing page with Solana-inspired neon aesthetics (purple/green gradients) and glassmorphism effects, plus a waitlist system and phased video editor development.

---

## Phase 1: Landing Page (Initial Build)

### 🎨 Design System Setup
- Dark theme with rich black background (#0D0D0D)
- Neon purple/green gradient accent colors (Solana-inspired)
- Glassmorphism card styles with blur effects
- Custom animations for hover effects and scroll reveals
- Futuristic sans-serif typography with glow effects

### 🚀 Hero Section
- Bold headline: "Edit Smarter. Publish Faster."
- Animated gradient background with subtle motion
- Prominent "Join the Waitlist" CTA button with neon glow
- Optional: Background video loop or animated UI snippets

### ⚡ Feature Showcase Grid
6 feature cards with neon-tinted icons:
- Real-Time Editing (live preview)
- Runs Locally (on-device AI)
- Auto-Subtitles (AI captions)
- Smart Clipping (scene detection)
- Audio Cleanup (noise reduction)
- Instant Export

Cards will have glass-effect backgrounds, neon borders, and subtle hover animations.

### 🖥️ Editor Demo Section
- Side-by-side mockup showing the editor UI
- Annotated with neon callouts (Timeline, Subtitle Panel, Audio Track)
- Static or lightly animated to simulate the editing experience
- "Instant preview" badge highlighting local AI speed

### 🔒 Why Local AI Section
- Privacy, Speed, and Cost benefits
- Animated chip/computer icon
- Clear messaging: "Your data never leaves your device"
- Visual diagram showing no cloud upload

### 💰 Pricing Section
Two pricing cards:
- **Starter** - $30/mo: Real-time editing, auto-subtitles, 5 projects/month
- **Pro** - $60/mo: Unlimited projects, smart audio, custom exports

Neon-styled cards with "Start Free Trial" buttons linking to waitlist.

### 💬 Testimonials Carousel
- 4-5 placeholder testimonials from "beta users"
- Dark glassmorphism cards with neon quote icons
- Smooth carousel navigation
- Optional: Faded partner logos below

### 📧 Waitlist Modal
- Email capture form triggered by CTAs
- Stored in Lovable Cloud database
- Success confirmation with animated feedback

### 🔗 Footer
- Clean minimal footer with muted grey links
- Sections: Docs, GitHub, Terms, Privacy
- Social icons: Twitter, Discord
- Subtle neon separator line

## Phase 2: Video Editor UI (Future Development)

Once the landing page is complete, we'll build the editor interface with:

### Timeline Editing
- Horizontal timeline with video/audio tracks
- Drag-and-drop clip management
- Trim and split functionality
- Playhead with scrubbing

### Video Preview
- Video player with playback controls
- Real-time preview during editing
- Fullscreen mode

### Auto-Subtitles Panel
- AI-generated captions display
- Edit/delete subtitle entries
- Timing adjustments

### Audio Tools
- Audio waveform visualization
- Noise reduction slider controls
- Volume adjustment per track

## Technical Approach
- React + TypeScript + Tailwind CSS
- shadcn/ui components styled with neon/glass theme
- Framer Motion for animations
- Lovable Cloud for waitlist database
- Recharts for any data visualizations
- Embla Carousel for testimonials
