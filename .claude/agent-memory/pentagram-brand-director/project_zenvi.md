**Product:** Zenvi — AI-powered video editing desktop app. Local processing (no cloud uploads), zero latency, AI captions in 50+ languages, smart scene detection, audio cleanup, instant export.

**Current brand state (pre-redesign):** Minimal high-contrast. Near-black (#0A0A0A) background, pure white text, single electric blue accent (#0066FF / hsl 216 100% 50%). No gradients, no glow, no glass. Inter typeface only. Logo is three staggered outlined squares (bottom-left to top-right).

**Target brand direction (post-redesign):** Space/interplanetary aesthetic. Deep space blacks, cosmic purples, nebula blues, stellar whites, aurora/nebula accent colors. Apple-level finish — minimalist, bold, sophisticated typography, refined interactions. Feels like looking into deep space with the precision and polish of Apple's design language.

**Tech stack:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion, Supabase (auth + database), Lucide icons.

**Key pages/components:** Landing (Hero, Navbar, Features, Comparison, Pricing, Footer, LogoTicker), Auth (Login, Signup, AuthCallback), Dashboard, Download, Checkout.

**Why:** Website redesign commissioned to elevate brand perception from "minimal developer tool" to "premium creative platform" in the interplanetary/cosmic design language.

**How to apply:** All brand decisions should be implementable in Tailwind CSS + CSS custom properties. Color values should include hex (for Tailwind config) and HSL (for CSS variables). Typography recommendations should use Google Fonts or commercially licensed fonts available via web. Motion principles should align with Framer Motion's spring/tween API.
