# Zenvi.pro

**Zenvi** is AI-powered video editing that runs entirely on your device. Edit smarter, publish faster—no uploads, no latency, no compromises.

## Overview

- **Local-first**: All processing happens on your machine; your videos never leave your device.
- **Real-time editing**: Live preview with instant changes and no rendering wait.
- **Privacy-focused**: No cloud uploads; captions, clipping, and enhancement run on-device.

## Tech stack

- [Vite](https://vitejs.dev/)
- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- [Supabase](https://supabase.com/) (e.g. waitlist)

## Getting started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

Then open the URL shown in the terminal (e.g. `http://localhost:8080`).
